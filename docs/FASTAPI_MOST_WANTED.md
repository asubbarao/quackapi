## Honest edges

quackapi does not (yet) solve everything. The genuinely open edges (verified 2026-07-09): **direct in-process TLS** is mechanism-limited (vendored mbedtls is crypto-only) so proxy termination is v1 — uvicorn's own answer; **inbound browser WebSockets** on the main port need C-server work (radio is client-only, outbound); **async I/O fan-out inside one handler** is conceded as different-not-equal; **multipart binary** v2 is an open call. Request-scoped DI setup/teardown is **shipped** (#11143) — the only residual is a dependency holding a *live object across statements*, rarely needed. gzip is **shipped** (in main, certified). The scoreboard above is only about the historic FastAPI "most wanted, closed unresolved" list — for the full current-state audit and the roadmap of what people want next, see [docs/STATUS.md](docs/STATUS.md).
